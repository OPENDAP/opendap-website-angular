Hyrax Data Server

Hyrax is the next generation server from OPeNDAP. It utilizes a modular design that employs a light weight Java servlet to provide the public-accessible client interface, and a back-end daemon, the BES, to handle the heavy lifting. The BES uses the same handlers that are used with Server3 (also know as the CGI Server) but loads those at run time.

Benefits of Hyrax

router

The servlet architecture is faster, more robust, and more secure than CGI invoked Perl scripts.

device_hub

A single installation can handle multiple data representations, such as hdf4, hdf5, and netcdf.

folder

THREDDS catalog functionality.
